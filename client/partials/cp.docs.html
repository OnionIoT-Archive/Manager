This simple tutorial will show you how to set up an Onion API to control a device over HTTP.

Materials

An internet-enabled Arduino (we prefer the Arduino Yun, but any Arduino with an Ethernet shield will work)

Jumper cables

Materials

Set up your Arduino to connect to the Internet using the Arduino Getting Started guide.

Connecting with Onion

Download the Onion example code

Set the device id generated in your Onion dashboard, for example, “d001”.

Notice in the setup() function in the example code how the get and post requests are set up to map to local Arduino functions.

You should now be able to control the LED by visiting

http://dev.onion.io/api/devices/ /on and

http://dev.onion.io/api/devices/ /on in your browser.
